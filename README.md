# FitTrack — Система учёта тренировок

FitTrack — веб-приложение для персональных тренеров. Позволяет вести учёт клиентов, пакетов тренировок и отдельных занятий.

## Возможности

- Регистрация и авторизация тренеров (с выбором специализации: фитнес / бег / плавание)
- Управление клиентами (добавление, редактирование, мягкое удаление)
- Создание пакетов абонементов с автогенерацией занятий по дням недели
- Учёт статуса каждого занятия: Запланировано / Отходил / Пропустил
- Дашборд с ключевыми метриками и уведомлениями о заканчивающихся пакетах

## Технологии

- **Backend:** Laravel 11, PHP 8.2+
- **Auth:** Laravel Breeze (кастомизированный)
- **БД:** MySQL (или SQLite для разработки)
- **Frontend:** Tailwind CSS (CDN), Font Awesome (CDN)

## Установка

### Требования

- PHP 8.2+
- Composer
- MySQL 8.0+ или SQLite
- Node.js (опционально, для Vite)

### Шаги

1. Клонируйте/скачайте проект:
   ```bash
   cd fittrack
   ```

2. Установите зависимости:
   ```bash
   composer install
   ```

3. Скопируйте `.env`:
   ```bash
   cp .env.example .env
   ```

4. Сгенерируйте ключ приложения:
   ```bash
   php artisan key:generate
   ```

5. Настройте базу данных в `.env`:
   ```
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=fittrack
   DB_USERNAME=root
   DB_PASSWORD=your_password
   ```
   
   Или для SQLite (проще для разработки):
   ```
   DB_CONNECTION=sqlite
   ```

6. Запустите миграции и сидер:
   ```bash
   php artisan migrate --seed
   ```

7. Запустите сервер разработки:
   ```bash
   php artisan serve
   ```

8. Откройте [http://localhost:8000](http://localhost:8000)

## Структура БД

| Таблица             | Описание                             |
|---------------------|--------------------------------------|
| `users`             | Тренеры (аутентификация Breeze)      |
| `clients`           | Клиенты тренера                      |
| `packages`          | Пакеты/абонементы клиента            |
| `training_sessions` | Отдельные занятия в рамках пакета    |

## Маршруты

| Метод  | URL                                | Действие                     |
|--------|------------------------------------|------------------------------|
| GET    | /dashboard                         | Дашборд                      |
| GET    | /clients                           | Список клиентов              |
| GET    | /clients/create                    | Форма создания клиента       |
| POST   | /clients                           | Сохранить клиента            |
| GET    | /clients/{id}                      | Карточка клиента             |
| GET    | /clients/{id}/edit                 | Редактировать клиента        |
| GET    | /clients/{id}/packages/create      | Форма нового пакета          |
| POST   | /clients/{id}/packages             | Сохранить пакет              |
| GET    | /packages/{id}/sessions            | Список занятий пакета        |
| POST   | /packages/{id}/sessions            | Добавить занятие             |
| PATCH  | /sessions/{id}                     | Обновить статус занятия      |
